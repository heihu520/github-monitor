## 📝 技术决策

### 为什么选择 Vite？
- 极快的冷启动
- 即时的模块热更新
- 真正的按需编译
- 优化的生产构建

### 为什么使用 Composition API？
- 更好的逻辑复用
- 更好的类型推导
- 更灵活的代码组织
- Vue 3的推荐方式

### 设计系统原则
- **一致性** - 统一的视觉语言和交互模式
- **可访问性** - 足够的对比度和清晰的视觉层次
- **响应性** - 适配所有设备尺寸
- **性能** - 优化动画和渲染性能

## ⚠️ 注意事项

1. **依赖安装** - 首次运行前必须执行 `npm install`
2. **Node版本** - 建议使用 Node.js 16+
3. **API配置** - 后端API地址在 vite.config.ts 中配置
4. **类型错误** - 安装依赖后TypeScript错误会消失
